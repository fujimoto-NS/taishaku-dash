# 貸借ダッシュ！

簿記3級の勘定科目について、借方・貸方の判断をランゲーム感覚で練習できるブラウザゲームです。

## GitHub Pages

このリポジトリは静的サイトとしてそのまま GitHub Pages で公開できます。

推奨設定:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

公開後のURL:

`https://fujimoto-ns.github.io/taishaku-dash/`

## Included Files

- `index.html`: ゲーム本体
- `404.html`: GitHub Pages用リダイレクト
- `og.svg`: X/OGP用画像
- `.nojekyll`: 静的配信設定
